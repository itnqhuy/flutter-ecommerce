# 🛍️ Tech Gear - Flutter eCommerce App

A tech product eCommerce app built using **Flutter**, featuring **light/dark theme** support and state management with **GetX**.

---

## 🚀 Key Features

- 🌗 Optional Light/Dark Theme  
- ⚡ State Management with GetX  
- 🔒 Login / Register / OTP Authentication  
- 🛍️ Product List, Product Details, Cart, Checkout  
- 🧑‍💼 Admin Web Interface for Management

## 🛠️ Technologies Used

- Flutter  
- Dart  
- GetX (state management, routing)  
- Firebase Auth  
- Cloud Firestore / Firebase Realtime Database  
- Responsive UI (Mobile + Web)

## 📦 Installation

```bash
git clone https://github.com/your-username/tech_gear_flutter.git
cd tech_gear_flutter
flutter pub get
flutter run
```

## 📌 Notes
- The app requires Firebase integration → configure the google-services.json file for Android and the GoogleService-Info.plist file for iOS.
- Make sure Authentication, Firestore, and Storage are enabled in the Firebase Console.
- If running on the web, you need to set up a Firebase web app and include the Firebase initialization script in index.html or your main web entrypoint.
